Tutorial

1. Selecting Data

When open the data visualizing page, the user have to select the tumor type, gene, data type and clinical information one by one.

Before completint the selection, no operation can be performed.

Do not afraid you make a mistake, the option can be edited afterward.

The first popupmenu (fig A) is about cancer type, please choose the cancer type you are interested and push selection buttom.

In the second popupmenu (fig B), please enter a Official Gene Symbol or Entrez ID. If the query term is not exist in the database, the alarm would be apear to inform you. Please check your input, or the gene is too new so that the analysing did not take count of it.

Then choose the Exon or Junction normalized expression (fig C), which to be display. More about the normalization method please refer to the next section or paper

At last choose the phenotype to display (fig D).

Tutorial

2. About graph

Data selection can adjust by clicking the corresponding item in the header bar.

The samples were sorted by clinical type and in each clinical type sorted by the gene expression. When the mouse hover the clinical row, the group name and a guide line will apear.

Each normalized exon expression value or junction expression value was plot by row and each samples was by column. A transient highlight of exon or junction pattern can be produced by mouse hover a typical exon or junction row. A stable highlight can be achieve by single click. What more, the double click of exon or junction graph will open a UCSC Genome Browsers in a new tab or windows with typical exon or junction area hightlight.

The Exon or Junction pattern is the exons and junctions that TCGA used to count the mapping reads. What's represented the only information that can be draw from the Level 3 data of the mRNA Sequencing datasets from TCGA.

Tutorial

3. Isoform expression

By clicking the transcriptional isoform, the boxplot of isoform expression between each group will show. At the top of the popup box, there are bottoms used to swhich the display of boxplot or km plot. At the top right side, you can see the ucsc id of the isoform, Click on the text will open a new tab or window display the UCSC Genome Browsers with the isoform hightlighted.

The km graph include four parts, the bottom right parts is the part used to adjust the cutpoint for grouping, the knob can be pull to change the cutpoint. The default value is the median. The top right part is display the grouping information include the cutpoint value and sample size of each group. The top left part display the survival line of each person that be included by defineing the surivival start time.

Flowchart and algorithms

The TCGA level 3 gene and isoform RSEM expression, exon and junction quantification value (UNC illuminaHiSeq_RNASeqV2), clinical data (Merged_clinical_level_1) were downloaded from TCGA ftp site -- Firehouse.

The R was used to generate transcriptional pattern and TCGA data clearup. The formated json data were import into mongodb.

The normalized exon and junction quantification value was used to visualization. The normalization like Normalized Index (NI) used in exon microarray analysis which accomplished by dividing sample's exon or junction quantificaiton value to the mean of exons or junctions quantification value of that gene. So that the gene expression variation can be wipe out and the normalized exon or junction expression level represented its usage.
