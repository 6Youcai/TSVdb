TSVdb

TCGA Splicing Variation DB

TSVdb is an easy-to-use web tool for the integration and visualization of mRNA splicing variation, transcirptional isoform expression and clinical information from The Cancer Genome Atlas project (TCGA) RNASeq data.

Tutorial

1. Selecting Data

When open the data visualizing page, the user must sequentially select the tumor type, gene, data type and clinical information.
Before completing the selection, no other operation can be performed.
Do not afraid when making a mistake in this stage, the options can be edited afterward.
The first popup menu (fig A) is on cancer type, please choose the cancer type you are interested and push selection button.
In the second popup menu (fig B), please enter an Official Gene Symbol or Entrez ID. If the query term does not exist in the database, the alarm would appear to notify you.
Then choose the scale Exon or Junction quantification value to be displayed (fig C). More about the scaling method please refer to the next section or paper
At last, choose the phenotype you are interesting (fig D).

Tutorial

2. About graph

Data selection can adjust by clicking the corresponding buttons in the header.
The samples were sorted by clinical type, then sorted by the gene expression. When the mouse hovers the clinical attribution row, the group name and a guide line will appear.
Each scaled exon quantification value or junction quantification value was display as area graph by row with samples by column. A transient highlight of exon or junction pattern generated when the mouse hovers an exon or junction area graph. A durable highlight can be achieved by a single click. What more, the double click of an exon or junction area graph will open the UCSC Genome Browsers in a new tab or windows with selecting exon or junction area highlight.
The Exon or Junction pattern is the exons and junctions that TCGA used to count the mapping reads. They represent the only information that can be drawn from the Level 3 data of the mRNA Seq datasets from TCGA.

Tutorial

3. Isoform expression

By clicking the transcriptional isoform, the box plot of isoform expression value (normalized RSEM) by each group appear. At the top of the pop-up box, there are buttons used to switch the display of box plot or KM plot. At the top right side, you can see the UCSC id of the isoform, Click on the text will open a new tab or window display the UCSC Genome Browsers with the isoform highlighted.
The KM graph includes four parts, the bottom right part is the part used to adjust the cutpoint for grouping, the knob can be pulled to change the cutpoint. The default value is the median. The top right part displays the grouping information includes the cutpoint value and sample size of each group. The top left part displays the survival line of each person that be included after filtering by the survival start time.

Flowchart and algorithms

The TCGA level 3 gene and isoform RSEM expression, exon and junction quantification value (UNC illuminaHiSeq_RNASeqV2), clinical data (Merged_clinical_level_1) were downloaded from TCGA ftp site -- Firehouse.
The R was used to generate transcriptional pattern and TCGA data cleanup. The formatted json data were import into MongoDB.
The scaled exon and junction quantification value were used to visualization. The scaling methods similar to Normalized Index (NI) used in exon microarray analysis which was accomplished by dividing sample's exon or junction quantification value to the mean of exons or junctions quantification value of that gene. So that the gene expression variation can be corrected and the scaled exon or junction expression level represented its usage.
